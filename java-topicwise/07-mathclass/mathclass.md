# ☕ Java Basics: Math Class

## 📌 Introduction

The **Math class** in Java provides methods and constants to perform mathematical operations such as square root, power, rounding, and more.
It is part of the **`java.lang` package**, so no import is required.

---

## 📦 What is Math Class?

The Math class is a **utility class** that contains:

* Static methods for mathematical calculations
* Mathematical constants

All methods are **static**, so they can be used directly without creating objects.

---

## 🔢 Mathematical Constants

### 🔍 Common Constants:

* `PI` → Represents the value of π (3.14159…)
* `E` → Represents Euler’s number (2.71828…)

### ⚡ Key Points:

* Used in scientific and geometric calculations
* Provide high precision values

---

## ⚙️ Important Methods

### 🔹 Power Function

* Used to raise a number to a power
* Returns result of exponentiation

---

### 🔹 Square Root

* Returns the square root of a number
* Used in geometry and formulas

---

### 🔹 Absolute Value

* Converts negative values to positive
* Keeps positive values unchanged

---

### 🔹 Rounding Functions

* **round** → Rounds to nearest integer
* **ceil** → Rounds up to nearest integer
* **floor** → Rounds down to nearest integer

---

### 🔹 Maximum and Minimum

* **max** → Returns the larger value
* **min** → Returns the smaller value

---

## 🎯 Features of Math Class

* No object creation required
* Fast and efficient
* Provides precise results
* Covers most common mathematical operations

---

## ⚠️ Important Notes

* All methods are static → accessed using class name
* Works with different data types (int, double, float, etc.)
* Some methods return double even if input is integer

---

## 🧠 Summary

* Math class is used for mathematical operations
* Part of `java.lang` package
* Contains constants like PI and E
* Provides methods like power, square root, rounding, max, and min

---

💡 *Tip:* Use Math class whenever you need built-in mathematical calculations instead of writing manual logic.
