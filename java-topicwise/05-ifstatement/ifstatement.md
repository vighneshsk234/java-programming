# ☕ Java Basics: `if` and `if-else` Statements

## 📌 Introduction

Decision-making statements in Java allow a program to execute different blocks of code based on conditions.
The most basic decision-making constructs are **`if`** and **`if-else`** statements.

---

## 🔹 `if` Statement

### 🔍 Definition:

The `if` statement is used to execute a block of code **only when a condition is true**.

### ⚡ Key Points:

* Condition must return a **boolean value** (`true` or `false`)
* If the condition is `true`, the block executes
* If the condition is `false`, the block is skipped
* Used when only one condition needs to be checked

---

## 🔸 `if-else` Statement

### 🔍 Definition:

The `if-else` statement is used to execute one block of code if the condition is true, and another block if the condition is false.

### ⚡ Key Points:

* Provides an alternative path of execution
* One block always executes (either `if` or `else`)
* Helps in handling two possible outcomes

---

## 🔁 Flow of Execution

1. Condition is evaluated
2. If condition is `true` → `if` block executes
3. If condition is `false` → `else` block executes

---

## ⚠️ Important Notes

* Conditions must be valid boolean expressions
* Curly braces improve readability and avoid errors
* Proper indentation makes code easier to understand

---

## 🎯 When to Use

* Use `if` when you want to execute something **only if a condition is true**
* Use `if-else` when you need **two possible outcomes**

---

## 🧠 Summary

* `if` → Executes block only when condition is true
* `if-else` → Chooses between two blocks
* Essential for decision-making in programs

---

💡 *Tip:* Always write clear and simple conditions to avoid logical errors.
