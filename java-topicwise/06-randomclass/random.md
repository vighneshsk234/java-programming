# ☕ Java Basics: Random Class

## 📌 Introduction

The **Random class** in Java is used to generate random values such as numbers and booleans.
It is part of the **`java.util` package** and is widely used in applications like games, simulations, and testing.

---

## 📦 What is Random Class?

The Random class provides methods to generate **pseudo-random values**.
These values appear random but are generated using a mathematical algorithm.

---

## 🎯 Features of Random Class

* Generates random integers, doubles, floats, and booleans
* Produces values within a specific range
* Can be used for simple randomness in programs
* Supports seeding for reproducible results

---

## 🔢 Types of Values Generated

* Integer values
* Floating-point values
* Boolean values
* Long values

---

## 🎲 Range of Values

* Random values can be generated within a specific range
* Default behavior may produce values within a large range
* Custom ranges can be defined as per requirement

---

## 🌱 Seed Concept

* A **seed** is an initial value used by the Random class
* Same seed → same sequence of random numbers
* Useful for testing and debugging

---

## ⚡ Key Points

* Random class generates **pseudo-random numbers**, not truly random
* Each method returns a different type of value
* Widely used in games, password generation, and simulations
* Easy to use and part of standard Java library

---

## ⚠️ Important Notes

* Not suitable for **secure applications** like encryption
* For security purposes, more advanced classes should be used
* Output depends on internal algorithm and seed value

---

## 🧠 Summary

* Random class is used to generate random values
* Part of `java.util` package
* Supports multiple data types
* Uses seed for generating sequences

---

💡 *Tip:* Use random values carefully when logic depends on unpredictability.
