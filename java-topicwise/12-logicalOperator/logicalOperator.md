# ☕ Java Basics: Logical Operators

## 📌 Introduction

Logical operators in Java are used to combine multiple conditions and return a boolean result (`true` or `false`).
They are mainly used in decision-making statements like `if`, `if-else`, loops, etc.

---

## 🔍 What are Logical Operators?

Logical operators work on **boolean expressions** and help in forming complex conditions.

---

## 🔤 Types of Logical Operators

### 🔹 Logical AND (`&&`)

* Returns `true` if **both conditions are true**
* Returns `false` if any one condition is false

---

### 🔹 Logical OR (`||`)

* Returns `true` if **at least one condition is true**
* Returns `false` only if both conditions are false

---

### 🔹 Logical NOT (`!`)

* Reverses the result of a condition
* `true` becomes `false`, and `false` becomes `true`

---

## 🔁 Short-Circuit Evaluation

### 🔹 AND (`&&`)

* If the first condition is `false`, the second condition is **not checked**

### 🔹 OR (`||`)

* If the first condition is `true`, the second condition is **not checked**

---

## ⚡ Key Points

* Used for combining multiple conditions
* Always returns a boolean result
* Improves decision-making logic
* Works with relational operators

---

## ⚠️ Important Notes

* Conditions must return boolean values
* Be careful with operator precedence
* Use parentheses to make logic clear

---

## 🎯 Real-Life Use Cases

* Checking multiple conditions (age and eligibility)
* Login validation (username and password)
* Input validation

---

## 🧠 Summary

* `&&` → Both conditions must be true
* `||` → At least one condition must be true
* `!` → Reverses condition
* Essential for building complex logic

---

💡 *Tip:* Use parentheses to avoid confusion in complex logical expressions.
