# ☕ Java Basics: String Methods

## 📌 Introduction

In Java, a **String** is a sequence of characters used to store text.
The `String` class provides many built-in methods to perform operations on text.

---

## 🔍 What are String Methods?

String methods are **predefined functions** that allow manipulation and analysis of strings.
They help in tasks like searching, modifying, comparing, and extracting data.

---

## 🔤 Common String Methods

### 🔹 Length

* Returns the number of characters in a string

---

### 🔹 Trim

* Removes leading and trailing spaces

---

### 🔹 Case Conversion

* Convert string to uppercase
* Convert string to lowercase

---

### 🔹 Comparison Methods

* Compare two strings for equality
* Case-sensitive and case-insensitive comparison

---

### 🔹 Contains

* Checks if a string contains a specific sequence of characters

---

### 🔹 Replace

* Replaces characters or words in a string

---

### 🔹 Substring

* Extracts a portion of a string

---

### 🔹 Index Of

* Finds position of a character or substring

---

### 🔹 Character Access

* Access a specific character using its index

---

### 🔹 Starts With / Ends With

* Checks if a string starts or ends with specific characters

---

### 🔹 Empty Check

* Checks if a string is empty

---

## ⚡ Key Points

* Strings in Java are **immutable** (cannot be changed after creation)
* Methods return new strings instead of modifying original
* String indexing starts from **0**

---

## ⚠️ Important Notes

* Case matters in string comparison
* Invalid index access can cause errors
* Always handle strings carefully to avoid runtime issues

---

## 🎯 Uses of String Methods

* Input validation
* Data processing
* Searching and filtering text
* Formatting output

---

## 🧠 Summary

* String methods help manipulate text efficiently
* Provide operations like searching, replacing, and extracting
* Essential for almost every Java program

---

💡 *Tip:* Mastering String methods will make your coding faster and more efficient.
