# ☕ Java Basics: Nested Loops

## 📌 Introduction

A **nested loop** is a loop inside another loop.
It is used when a program needs to perform repetitive tasks **within another repetitive task**.

---

## 🔍 What is a Nested Loop?

* A loop placed **inside another loop**
* The inner loop executes **completely for each iteration** of the outer loop

---

## 🔁 Flow of Execution

1. Outer loop starts execution
2. Inner loop runs completely
3. Outer loop moves to next iteration
4. Inner loop runs again
5. This process continues until outer loop ends

---

## 🎯 Example Concept

If outer loop runs 3 times and inner loop runs 2 times,
then inner loop executes **3 × 2 = 6 times**

---

## ⚙️ Types of Nested Loops

* `for` inside `for`
* `while` inside `while`
* `for` inside `while` (and vice versa)

---

## 📊 Uses of Nested Loops

* Pattern printing (stars, numbers)
* Matrix operations
* Table generation
* Multi-dimensional data processing

---

## ⚡ Key Points

* Inner loop depends on outer loop
* Total iterations = outer × inner
* Helps in solving complex repetitive tasks

---

## ⚠️ Important Notes

* Too many nested loops can reduce readability
* Can increase time complexity
* Proper indentation is important

---

## 🧠 Summary

* Nested loop = loop inside another loop
* Inner loop runs fully for each outer loop iteration
* Useful for patterns and multi-level repetition

---

💡 *Tip:* Always trace nested loops step-by-step to understand execution clearly.
