# ☕ Java Basics: `while` Loop

## 📌 Introduction

The `while` loop in Java is used to execute a block of code **repeatedly** as long as a condition is true.
It is useful when the number of iterations is **not known in advance**.

---

## 🔍 What is `while` Loop?

* A **control flow statement**
* Repeats execution based on a condition
* Checks the condition **before** executing the loop body

---

## 🔁 Flow of Execution

1. Condition is evaluated
2. If condition is `true` → loop body executes
3. After execution, condition is checked again
4. Loop continues until condition becomes `false`

---

## ⚙️ Key Features

* Entry-controlled loop (condition checked first)
* May execute **zero or more times**
* Simple and easy to use

---

## 🎯 When to Use `while`

* When number of iterations is unknown
* When loop depends on user input
* When condition-based repetition is needed

---

## ⚡ Key Points

* Condition must return a boolean value
* Loop stops when condition becomes false
* Useful for input validation and repeated tasks

---

## ⚠️ Important Notes

* If condition never becomes false → infinite loop
* Always update loop variable inside loop
* Proper condition is necessary to avoid errors

---

## 🧠 Summary

* `while` loop repeats code while condition is true
* Condition is checked before execution
* Can execute zero or multiple times
* Used for condition-based repetition

---

💡 *Tip:* Always ensure your loop condition will eventually become false to avoid infinite loops.
