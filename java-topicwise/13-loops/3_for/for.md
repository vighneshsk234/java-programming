# 🔁 Java for Loop (Explanation)

The `for` loop is a control structure used to execute a block of code **a specific number of times**. It is most commonly used when the number of iterations is known in advance.

---

## 📌 How it Works

The `for` loop combines three important parts in a single line:

1. **Initialization** → starting point of the loop
2. **Condition** → determines whether the loop continues
3. **Update** → changes the loop variable after each iteration

---

## 🔄 Key Characteristic

👉 The `for` loop checks the condition **before execution**, so it may execute **zero or more times**.

---

## 🧠 When to Use

Use a `for` loop when:

* The number of iterations is **known**
* You are working with **ranges or sequences**
* You need a **counter-controlled loop**
* You are iterating over arrays or collections

---

## ⚙️ Execution Flow

1. Initialization is executed once
2. Condition is checked
3. If true → execute loop body
4. Update is performed
5. Repeat from step 2
6. If condition is false → exit loop

---

## ⚠️ Important Points

* All three components (initialization, condition, update) are written in one line
* Any part can be omitted, but semicolons must remain
* Incorrect conditions may cause an **infinite loop**
* Loop variable scope is usually **limited to the loop**

---

## 🔍 Difference from Other Loops

* More compact than `while` loop
* Best suited for **count-based iteration**
* Less flexible than `while` when conditions are complex
