# ☕ Java Basics: Operators and Expressions

## 📌 Introduction

Operators in Java are symbols used to perform operations on variables and values.
Understanding operators is essential for writing logical and mathematical expressions.

---

## ➕ Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations.

### 🔍 Types of Arithmetic Operations:

* Addition → `+`
* Subtraction → `-`
* Multiplication → `*`
* Division → `/`
* Modulus (remainder) → `%`

### ⚡ Key Points:

* Division between integers gives integer result (decimal part removed)
* Modulus operator gives remainder
* Arithmetic operators follow precedence rules

---

## 🔁 Augmented Assignment Operators

Augmented assignment operators combine arithmetic operation with assignment.

### 🔍 Common Operators:

* `+=` → Add and assign
* `-=` → Subtract and assign
* `*=` → Multiply and assign
* `/=` → Divide and assign
* `%=` → Modulus and assign

### ⚡ Key Points:

* Shorter and more efficient form of writing expressions
* Improves code readability
* Performs operation and assignment in a single step

---

## ⬆️ Increment Operator (`++`)

Used to increase the value of a variable by 1.

### 🔍 Types:

* **Pre-increment (`++x`)**

  * Value is increased first, then used

* **Post-increment (`x++`)**

  * Value is used first, then increased

### ⚡ Key Difference:

* Pre-increment → Change happens before use
* Post-increment → Change happens after use

---

## ⬇️ Decrement Operator (`--`)

Used to decrease the value of a variable by 1.

### 🔍 Types:

* **Pre-decrement (`--x`)**

  * Value is decreased first, then used

* **Post-decrement (`x--`)**

  * Value is used first, then decreased

### ⚡ Key Difference:

* Pre-decrement → Change happens before use
* Post-decrement → Change happens after use

---

## 📊 Order of Operations (Operator Precedence)

Operator precedence defines the order in which operations are performed in an expression.

### 🔝 Precedence Order (High → Low):

1. Parentheses `( )`
2. Unary operators (`++`, `--`)
3. Multiplication, Division, Modulus (`*`, `/`, `%`)
4. Addition, Subtraction (`+`, `-`)
5. Assignment operators (`=`, `+=`, `-=`, etc.)

### ⚡ Key Points:

* Expressions are evaluated based on precedence rules
* Parentheses can be used to change the order
* Operators with same precedence are evaluated left to right (associativity)

---

## ⚠️ Important Notes

* Misunderstanding precedence can lead to wrong results
* Always use parentheses for clarity when needed
* Increment and decrement operators behave differently based on position

---

## 🧠 Summary

* Arithmetic operators perform calculations
* Augmented operators simplify assignments
* Increment/Decrement modify values by 1
* Operator precedence controls evaluation order

---

💡 *Tip:* When in doubt, use parentheses to make expressions clear and avoid mistakes.
